## 一、项目概述

### 1.1 产品定义

OpenClaw Manager 是 OpenClaw 的图形化管理桌面应用，为 OpenClaw 提供零命令行的配置、管理和监控界面。

### 1.2 项目目标

| 目标 | 状态 | 说明 |
|------|------|------|
| 零命令行操作 | ✅ 达成 | 全程图形界面 |
| 向导式配置 | ✅ 达成 | 4 步 Agent 创建向导，首次引导 |
| 实时监控 | ✅ 达成 | 60 秒自动刷新，Gateway 状态实时显示 |
| 安全可靠 | ✅ 达成 | 备份 AES-256-GCM 加密 |

### 1.3 目标用户

- OpenClaw 用户（开发者、AI 爱好者）
- 不熟悉命令行的用户
- 需要可视化管理多个 Agent 的用户

## 六、部署说明

### 6.1 安装部署

1. 分发 `openclaw-manager.exe` 安装包
2. 用户双击安装
3. 首次启动自动检测环境

### 6.2 源码部署

```bash
git clone https://github.com/husir124/openclaw-manager.git
cd openclaw-manager
pnpm install
pnpm tauri build
```

### 6.3 开发环境

```bash
pnpm install
pnpm tauri dev
```
